This was a group project where we made a chatbot that would ask a number of questions to a patient and provide a prediction on what illness, disease, or issue they might have. It would also return potential tests for further analysis or even treatements. 

This project was aimed to be used by doctors to aid them while dealing with patients. Of course, as this was a student project, it was not aimed to be released to the public. Additionally, the data scraped was only in a certain time frame and with medicine the data is continously changing 

I specifically worked on the final part of this project. I connected the finalized classifier to the llm and had it take treatements and tests from a database that I had helped create by scraping the NHS's and other medical websites.
